# サイトマップ：みんなの本棚

1. **Top (Welcome Page)**
   - アプリ紹介 / 6分読書の啓蒙
   - 「ホーム画面に追加」バナー
   - [本棚を見る] [本を登録する（管理者）]

2. **Shelf (本棚一覧)**
   - 蔵書グリッド表示（貸出中/利用可能）
   - 簡易フィルタ（検索）

3. **Book Detail (本の紹介)**
   - 書籍詳細 / 寄贈者・他者レビュー
   - [貸出申請へ] ボタン

4. **Loan Form (貸出申請フォーム)**
   - 名前入力（LocalStorageより補完）
   - 返却予定日選択（カレンダー形式）
   - [確定] ボタン

5. **Return & Review (返却・レビュー)**
   - ※本棚の返却QRから遷移
   - 借りている本の選択
   - レビュー入力 ＋ [返却完了] ボタン

6. **Admin (管理者ページ)**
   - ISBNスキャン（蔵書登録）
   - 既存データの編集・削除
