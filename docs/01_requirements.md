# 要件定義書：みんなの本棚

## 1. プロジェクト概要
個人の所持本を寄贈・共有するミニ図書館のための、インストール不要なPWA。
「6分間の読書でストレスを68%減少させる」をテーマに、読書へのハードルを下げることを目的とする。

## 2. 機能要件
- **蔵書登録**: ISBNスキャン（quagga2）によりGoogle Books APIから情報を自動取得。
- **本棚閲覧**: 蔵書一覧の表示、貸出中ステータスの可視化（返却予定日の表示）。
- **貸出申請**: 会員登録不要。名前（LocalStorage保存）と返却予定日（最大2週間）の入力。
- **返却機能**: 本棚の「返却用QR」スキャンによる簡易返却フローとレビュー投稿。
- **通知機能**: PWAプッシュ通知による返却リマインド。
- **管理者機能**: 特定URL/合言葉による蔵書データの編集・削除。

## 3. 非機能要件
- **PWA**: オフラインキャッシュ、ホーム画面追加バナーの表示。
- **UX**: モバイルファースト、6分読書を促すメッセージの随所配置。
- **セキュリティ**: 会員登録を排した匿名性を重視しつつ、LocalStorageによる利便性確保。
